                   AERMAP README (dated 18081)              04/23/2018
                      
This file describes the components included in the zip files for the 
AERMAP terrain preprocessor (dated 18081) for the AERMOD
dispersion model. 

The following lists files associated with this zip
to the AERMAP terrain preprocessor (note that the AERMAP test cases have
not been updated with version 11103 given the limited nature of the 
code modifications associated with version 11103):

FILENAME                DESCRIPTION
-------------------	    ----------------------------------------------

aermap.exe              The AERMAP executable. The AERMAP executable was 
                        generated by the gfortran open source Fortran 
                        compiler.

alaska.las              The grid shift files needed to convert 7.5-minute 
                        DEM data to NAD83 for Alaska, US.

alaska.los              The grid shift files needed to convert 7.5-minute 
                        DEM data to NAD83 for Alaska, US.

conus.las               The grid shift files needed to convert 7.5-minute 
                        DEM data to NAD83 for Alaska, US.

conus.los               The grid shift files needed to convert 7.5-minute 
                        DEM data to NAD83 for the contiguous U.S. (CONUS).

hawaii.las              The grid shift files needed to convert 7.5-minute 
                        DEM data to NAD83 for contiguous U.S. (CONUS).

hawaii.los              The grid shift files needed to convert 7.5-minute 
                        DEM data to NAD83 for Hawaii, US.

prvi.las                The grid shift files needed to convert 7.5-minute 
                        DEM data to NAD83 for Puerto Rico/Virgin Islands, 
                        US.

prvi.los                The grid shift files needed to convert 7.5-minute 
                        DEM data to NAD83 for Puerto Rico/Virgin Islands,  
                        US.

stgeorge.las            The grid shift files needed to convert 7.5-minute 
                        DEM data to NAD83 for the Alaskan Island Group St. 
                        George, US.

stgeorge.los            The grid shift files needed to convert 7.5-minute 
                        DEM data to NAD83 for the Alaskan Island Group St. 
                        George, US.

stlrnc.las              The grid shift files needed to convert 7.5-minute 
                        DEM data to NAD83 for the Alaskan Island Group St. 
                        Lawrence, US.

stlrnc.los              The grid shift files needed to convert 7.5-minute 
                        DEM data to NAD83 for the Alaskan Island Group St. 
                        Lawrence, US.

stpaul.las              The grid shift files needed to convert 7.5-minute 
                        DEM data to NAD83 for the Alaskan Island Group St. 
                        Paul, US.

stpaul.los              The grid shift files needed to convert 7.5-minute 
                        DEM data to NAD83 for the Alaskan Island Group St. 
                        Paul, US.
